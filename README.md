# D-LLM — Diffusion LLMs from Scratch

A beginner-friendly resource for understanding and (eventually) building Diffusion Language Models — written for people who already know how AR LLMs (like GPT) work.

> There's no "nanoGPT" equivalent for Diffusion LLMs yet. This is an attempt to change that.

## Who Is This For

- You understand how GPT/AR LLMs work
- You want to understand what Diffusion LLMs do differently
- You want to build one from scratch, or help someone who is
